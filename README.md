# TogetherInCrisis – Volunteer & Crisis Response Platform

## 📌 Overview

**TogetherInCrisis** is a web-based platform designed to connect volunteers, NGOs, and organizations during emergencies and community service initiatives. The platform enables individuals to register as volunteers, discover volunteering opportunities, participate in crisis-response activities, and contribute to social causes effectively.

The system streamlines communication, volunteer management, and event coordination, helping communities respond faster and more efficiently during crises such as natural disasters, pandemics, and humanitarian emergencies.

## 💻 Installation & Setup

### Prerequisites

* Node.js
* MySQL
* Git

### Clone the Repository

```bash
git clone https://github.com/your-username/together-in-crisis.git
cd together-in-crisis
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=togetherincrisis
```

### Run the Application

```bash
npm start
```

For development:

```bash
npm run dev
```
